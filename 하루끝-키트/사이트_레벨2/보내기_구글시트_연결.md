# "보내기" → 구글시트 연결 (고객이 적은 게 시트에 쌓이게)

> 지금은 고객이 [보내기]를 눌러도 아무 데도 안 감(ENDPOINT 비어 있음).
> 아래대로 하면, 보낸 내용이 **윤미선 구글시트에 한 줄씩** 쌓임. 사업자등록·결제 필요 없음. 무료.
> 윤미선이 ①~④ 하고 **주소(URL)를 AI에게 주면**, AI가 사이트에 붙이고 → 다시 배포.

## ① 빈 구글시트 만들기
- 브라우저에서 **sheets.new** 입력 → 빈 시트 생성.
- 이름: "하루끝 응답" 정도로.

## ② 앱스스크립트 열기
- 그 시트에서 위 메뉴 **확장 프로그램 → Apps Script**.
- 열린 편집기의 코드를 다 지우고, 아래를 통째로 붙여넣기:

```javascript
function doPost(e){
  try{
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = ss.getSheets()[0];
    var data = JSON.parse(e.postData.contents);
    var headers;
    if (sheet.getLastRow() === 0){
      headers = ['받은시각'].concat(Object.keys(data));
      sheet.appendRow(headers);
    } else {
      headers = sheet.getRange(1, 1, 1, sheet.getLastColumn()).getValues()[0];
      Object.keys(data).forEach(function(k){
        if (headers.indexOf(k) === -1){
          sheet.getRange(1, headers.length + 1).setValue(k);
          headers.push(k);
        }
      });
    }
    var row = headers.map(function(h){
      if (h === '받은시각') return new Date();
      return data[h] !== undefined ? data[h] : '';
    });
    sheet.appendRow(row);
    return ContentService.createTextOutput(JSON.stringify({ok:true}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err){
    return ContentService.createTextOutput(JSON.stringify({ok:false, error:String(err)}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
```

- 위쪽 💾(저장) 누르기.

## ③ 웹앱으로 배포
- 오른쪽 위 **배포 → 새 배포**.
- 톱니바퀴(유형 선택) → **웹 앱**.
- 설정:
  - 실행 계정(Execute as): **나(me)**
  - 액세스 권한(Who has access): **모든 사용자(Anyone)**
- **배포** 누르기 → "액세스 승인" 뜨면 본인 구글 계정으로 허용(처음엔 "안전하지 않음"처럼 보여도 본인 스크립트라 괜찮음 → 고급 → 이동).
- 끝나면 **웹 앱 URL**이 나와요. `https://script.google.com/macros/s/...../exec` 형태.
  **이 URL을 복사해서 AI에게 주세요.**

## ④ AI가 할 일 (URL 받은 뒤)
- `기본.html` 의 `var ENDPOINT = "";` 에 그 URL을 넣음.
- 윤미선이 사이트 다시 배포(`vercel --prod`).
- → 이제 고객이 [보내기] 누르면 시트에 줄줄이 쌓임.
