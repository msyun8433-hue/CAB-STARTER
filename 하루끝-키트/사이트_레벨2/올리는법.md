# 사이트 인터넷에 올리기 (배포) — 윤미선이 직접 누르는 단계

> AI는 "인터넷 게시"를 직접 못 누른다(안전 규칙 §10). 아래는 **윤미선이 터미널에 붙여넣는** 절차.
> 이미 Vercel 로그인 돼 있음(msyun8433-hue). 한 번만 설정하면, 다음부터는 명령 한 줄로 갱신.

## 처음 올릴 때
1. VS Code 위 메뉴 **Terminal → New Terminal**.
2. 아래 두 줄을 붙여넣고 Enter:
   ```
   cd "/Volumes/NO NAME/cab-starter/하루끝-키트/사이트_레벨2"
   vercel --prod
   ```
3. 묻는 말에 이렇게 답:
   - **Set up and deploy?** → 그냥 Enter (yes)
   - **Which scope?** → Enter (내 계정)
   - **Link to existing project?** → `n` Enter
   - **What's your project's name?** → `trace` 입력 Enter  *(영문으로!)*
   - **In which directory is your code located?** → 그냥 Enter (`./`)
   - **Want to modify settings?** → `n` Enter
4. 1~2분 기다리면(음성 업로드) 끝에 이렇게 떠요:
   `✅ Production: https://trace.vercel.app` ← **이게 링크예요.** 폰에서 열어보기.

## 다음부터 (음성·내용 바꾼 뒤 다시 올리기)
같은 폴더에서 이 한 줄만:
```
vercel --prod
```
→ **링크는 그대로, 내용만 갱신**돼요. (친구한테 준 링크 안 죽음)
