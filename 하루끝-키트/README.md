# 하루끝-키트 — 셀프 세션 키트 제작 작업방

> 첫 도구 **〈하루 끝, 나에게〉** 를 만드는 곳.
> 완성품 1개를 만드는 게 아니라, **하루 끝 "상태"를 넣으면 셀프 세션 초안이 나오는 제작 흐름**을 실험한다.
> 작업 기준(윤미선 결·금지어)은 폴더 맨 위 `CLAUDE.md` 를 따른다. 현재 방향은 `자료함/★지금-기준.md`.

---

## 어떻게 굴러가나 (한 줄 흐름)
**입력(상태) + 씨앗(내 결) + 틀** 을 → 조립 → **초안 생성** → **비교·평가** → 다음 버전

## 방 7개 (각 폴더 안 `_여기엔.md` 에 설명 있음)
| 폴더 | 무엇 |
|---|---|
| `seeds/` | 윤미선 씨앗 — 브랜드 결·톤·금지어 (Day 2) |
| `inputs/` | 하루 끝 "상태" 입력 (예: 나와 멀어진 밤) (Day 3) |
| `prompts/` | 생성용 프롬프트 (제로베이스용 / 씨앗 반영용) |
| `templates/` | 세션 출력 틀 (Day 3) |
| `outputs/` | 만들어진 세션 초안 (Day 4~5) |
| `evaluation/` | 비교표·평가기준·주차 보고서 (Day 6~7) |
| `scripts/` | 조립용 작은 프로그램 (2주차) |

## 1주차 핵심 질문
제로베이스 초안 vs 윤미선 씨앗 반영 초안 — **어느 쪽이 "혼자 쓸 수 있는 셀프 세션"에 더 가까운가?**
(이번 주는 완성하는 주가 아니라 **비교하는 주**.)
