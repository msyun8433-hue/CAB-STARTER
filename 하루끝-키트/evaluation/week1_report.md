# 1주차 보고서 — 〈하루 끝, 나에게〉
작성: 2026-06-21 (2주차 수업 제출용) · 발표용으로 그대로 읽어도 됨

## 1. 내가 고른 자동화 영역
하루 끝에 **자기 자신과 대화하는 "셀프 세션"** 을 만드는 흐름.
(질문 생성기나 기록 정리 비서가 아니라 — 오디오·질문·기록지가 하나로 이어지는, 혼자 쓰는 도구)

## 2. 목표 한 문장
> 하루 끝에 자기 자신과의 대화를 다시 이어갈 수 있도록, 오디오·문장·질문·기록지가 하나의 흐름으로 이어지는 셀프 세션을 만든다.

## 3. 원래 계획 vs 실제로 한 것 (솔직하게)
- **원래 1주차 계획:** 상태를 입력하면 키트 초안이 자동 생성되는 "제작 엔진"을 만들고, *제로베이스 초안 vs 내 씨앗 반영 초안*을 나란히 만들어 **비교하는 실험.** (완성 상품은 만들지 말 것)
- **실제로 한 것:** 비교 실험 대신, **첫 키트 하나를 끝까지 만들어 인터넷에 출시**함.
- **왜 바꿨나:** 내 1순위 문제가 "탐색만 길고 출시가 늦는 것"이라, 실험보다 *손에 잡히는 첫 도구를 띄우는 것*을 택함. (평일 직장으로 시간이 묶여 있어, 출시로 좁히는 게 맞다고 판단)

## 4. 실제로 만든 것 (보여줄 것)
- **인터넷 링크(라이브):** https://harukkeut-site.vercel.app — Claude Code로 만들고 Vercel에 배포.
- **흐름:** "이어, 나와 어울릴까?" 1분 게임 → 결과(%) → 첫 세션(무료)
- **첫 세션:** 내 목소리 오디오(약 11분, 앞 잡음 잘라냄) + 화면 워크북(감각·오래 남은 장면·두 목소리·오늘의 한 문장·내일의 작은 이어감·감사 3가지) + PDF·Word 다운로드 + "보내기" → 내 지메일로 도착
- **Claude Code로 직접 해결한 실제 문제들:** 한글 파일명이 인터넷에서 404 → 영어로 교체 / 빈칸 많아 압박돼 보임 → 접기로 여백 살림 / 메일 주소 노출 → 인코딩으로 숨김 / 오디오 앞 잡음 → 3초 잘라 재배포 / 브라우저 캐시 → 주소에 버전 표시
- **이 밖에:** 내 사업을 8영역으로 펼친 지도(`business/02-8부서.md`) + CLAUDE.md(내 사업 매뉴얼) + 노션 MCP 연결까지 했다.

## 5. 이번 주에 배운 것 · 내가 내린 결정
- **오디오는 조미료가 아니라 핵심** — 워크북을 읽어주는 게 아니라, 눈 감고 데려가는 "세션". (안 그러면 "워크북만 있으면 되지 왜 들어?"가 됨)
- **명상·소마틱이어도 그 단어를 안 쓴다** — 일어나는 일 그대로만(숨·몸·머무름). '상담'도 안 꺼냄.
- **게임형 진단이 "사보세요"보다 부담을 낮춘다** — 통증형 + 이미 자기와 대화하는 형(결이 맞는 형) 둘 다 높게 나오게 설계.
- **안전 규칙** — 웹 배포는 AI가, 메일·결제는 내가 마지막에 누름.

## 6. 안 한 것 (솔직하게)
- 계획서의 "제작 엔진 + 제로베이스/씨앗 비교 실험 + 비교표·평가표"는 안 함. (대신 완성 제품을 출시함)
- 폴더 이름 변경(예정: "이어"), GitHub 로그인 — 아직.
- MCP는 노션만 연결. 강의에서 권한 5개(supabase·github·google·playwright)는 필요해질 때 붙일 예정.

## 7. 2주차에 진짜로 물어볼 것 (태오에게 — 우리끼리는 못 정하는 것)
1. **제품 하나 vs 반복 제작 시스템 — 뭘 먼저?**
   저는 1주차에 *완성 제품 하나*를 출시했는데, 강의 방향은 "반복 제작 엔진(시스템)"이었어요.
   1인 사업이 **제 몸이 안 갈리고** 굴러가려면, 제품을 더 내는 것보다 *시스템(엔진)을 먼저* 만드는 게 맞나요?
   제가 진짜 핵심(시스템)을 건너뛴 건가요, 아니면 제품을 먼저 낸 게 오히려 맞는 순서인가요?
2. **제 무기를 Claude Code로 자동화하는 게 어느 정도 규모인가요?**
   제 진짜 무기는 "AI가 고객이 쓴 글을 되비춰주고, 쌓이면 한 권이 되는 루프"예요.
   이걸 실제로 자동화하는 게 **4주 안에 작게라도 가능한** 일인가요, 아니면 어디서부터 손대야 하나요?
   (사람이 안 끼고 굴러가게 만드는 최소 단위가 뭘까요?)
3. **시간이 묶인 사람의 '자동화 1개'는 무엇으로 보시나요?**
   평일 직장으로 하루 8~10시간이 묶여 있어요. 제 노동을 **가장 많이 줄여줄 자동화 딱 하나**를 고른다면,
   제 상황(혼자·시간 적음·도구 판매)에서 뭘 보시겠어요?
