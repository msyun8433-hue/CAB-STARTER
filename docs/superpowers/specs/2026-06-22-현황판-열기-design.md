# 현황판 더블클릭 열기 — 설계 문서

> 작성: 2026-06-22 · 상태: 검토 대기
> 관련 원칙: CLAUDE.md §0(도구가 일하는 구조·몸 안 갈림), §8.5(USB로 기기 간 이동), §13(스킬 충실)

## 1. 목적 (왜 만드나)

현황판(`하루끝-키트/현황판.html`)은 정적 HTML이라 그냥 열면 옛 내용이 박제돼 있다.
"항상 최신"을 보려면 열기 전에 `node automations/session-dashboard.mjs` 로 다시 그려야 한다.
이 두 단계(갱신 → 열기)를 **더블클릭 한 번**으로 합친다.

성공 기준: 파일 하나 더블클릭 → 현황판이 **최신 상태로** 브라우저에 뜬다. (현황판.html 을 직접 더블클릭하던 수고와 동일, 대신 항상 최신)

## 2. 안 하는 것 (YAGNI)

- 자동 예약(매일 정해진 시각 실행) — USB·기기이동과 안 맞아 접음
- 로컬 서버 상시 구동 — 서버 켜두기 부담, USB 빼면 멈춤
- 커스텀 아이콘·설치 마법사 없음
- HTML 자체의 자동 갱신 — 브라우저 보안상 file:// HTML은 폴더를 못 읽어 불가능(그래서 node 가 필요)

## 3. 만들 것 — 더블클릭 실행 파일 2개

프로젝트 **루트**에 둔다(폴더 열면 바로 보이게):
- `현황판_열기.command` — 맥북용
- `현황판_열기.bat` — 회사 윈도우용

각 파일이 하는 일(동일):
1. 자기 파일이 있는 폴더로 이동
2. `node automations/session-dashboard.mjs` 실행(현황판 갱신)
3. `하루끝-키트/현황판.html` 을 기본 브라우저로 열기

## 4. 핵심 — USB 따라다녀도 작동 (이식성)

절대경로(`/Volumes/NO NAME/...`)를 **쓰지 않는다.** USB가 어느 기기·어느 드라이브로 꽂혀도 작동해야 하므로, 각 스크립트는 **자기 파일 위치 기준**으로 경로를 잡는다:
- 맥(`.command`): `cd "$(dirname "$0")"` 로 스크립트 폴더로 이동
- 윈도우(`.bat`): `cd /d "%~dp0"` 로 배치 파일 폴더로 이동

그 뒤 상대경로(`automations/session-dashboard.mjs`, `하루끝-키트/현황판.html`)로 실행·열기.

## 5. 오류 처리

- **node 없을 때:** 조용히 실패하지 않는다. "node(실행 엔진)가 필요해요. 설치 후 다시 시도하세요." 메시지를 띄우고 멈춘다.
  - 맥: `command -v node` 로 확인 후 없으면 echo + `read`(창 유지)
  - 윈도우: `where node` 로 확인 후 없으면 echo + `pause`
- **갱신 실패 시(node 실행 에러):** 에러 메시지가 창에 보이게 두고, 그래도 기존 현황판.html 은 열어준다(아주 옛것이라도 빈 화면보단 낫다). 단 node 자체가 없으면 위에서 멈춤.

## 6. 동작 방식 상세

- 맥 `.command` 는 실행 권한이 있어야 더블클릭으로 돈다 → 만든 뒤 `chmod +x` 적용.
- 맥에서 USB의 `.command` 첫 더블클릭 시 Gatekeeper("확인되지 않은 개발자") 경고가 날 수 있음 → 첫 1회만 우클릭→열기. (이건 코드로 못 없애고 사용 안내로 처리)
- 윈도우 `.bat` 는 실행 권한 개념이 없어 더블클릭 즉시 동작.

## 7. 검증

- **맥:** 실제로 `./현황판_열기.command` 실행 → 갱신 로그 뜨고 현황판.html 이 최신으로 열리는지.
- **맥(이식성):** 스크립트가 절대경로 없이 자기 위치 기준으로 도는지 코드로 확인.
- **윈도우 `.bat`:** 맥에서는 실행 불가 → 문법·경로(`%~dp0`, 상대경로, `where node`, `start`)를 코드 리뷰로 점검. 실제 더블클릭 검증은 회사 PC에서 윤미선이 1회 확인(안 되면 그때 고침).
- node 없을 때 안내 메시지가 나오는지(맥에서 PATH 비워 모의 확인 가능하면).

## 8. 비고

- 이 두 파일은 사이트 배포 대상이 아니다(폴더 안 도구). `.vercelignore` 등과 무관.
- 윤미선은 이제 현황판을 볼 때 `현황판.html` 대신 **`현황판_열기`** 를 더블클릭하면 된다.
