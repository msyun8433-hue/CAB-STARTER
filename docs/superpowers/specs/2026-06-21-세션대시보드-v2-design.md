# 세션 대시보드 v2 — 설계 메모

- 날짜: 2026-06-21
- 절차: superpowers:brainstorming (질문 → 설계 → 승인 완료)

## 목적
v1(진행상황 + 다음 단계)에 더해, **잊힌/잠든 세션을 깨워준다.** 각 항목이 마지막으로 손댄 날과 며칠째 멈췄는지 보여, 오래 묵힌 걸 놓치지 않게.

## v1에서 더해지는 것
1. 각 항목에 **날짜 배지** — "M/D 손댐 · N일째".
2. **오래 멈춤(7일+) 강조 + 위로** — 색을 다르게, 각 칸 맨 위로 정렬.
3. **잠든 세션 알림** — 7일+ 멈춘 게 있으면 맨 위 한 줄.
4. v1의 *다음 한 걸음 · 진행 보드 · 쌓인 것* 은 그대로.

## 방법
- "마지막 손댄 날"은 **파일 수정일(mtime)** 로 자동 산출. 수동 입력 없음.
- 멈춤 대상: **재료 · 대본 초안**(다음 행동이 필요한 것). 녹음 완료는 '잠듦'으로 안 봄(이미 끝).
- 기준: 7일.

## 산출물 (기존 v1은 그대로 둠)
- 생성기: `automations/session-dashboard-v2.mjs`
- 결과: `하루끝-키트/대시보드-v2.html`
- 갱신: `node automations/session-dashboard-v2.mjs`

## 안 하는 것 (YAGNI)
- 수동 날짜 입력 · 실시간 서버 · 복잡한 차트.
