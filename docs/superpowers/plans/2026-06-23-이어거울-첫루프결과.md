## 사용한 샘플 (베타 기록 1개)
```
하루 끝, 나에게 — 민
3. 감각: 무거움, 어깨가 뻐근함 / 한 단어: 버팀
4. 오래 남은 장면: 회의에서 내 의견이 묻혔다 / 그때 몸의 감각: 가슴이 조임 / 장면이 말하는 것: 또 참았네
5. 목소리 A: 괜찮은 척한다 / 목소리 B: 사실 인정받고 싶다
6. 오늘의 한 문장: 오늘의 나는 잘 버텼다 / 느낌: 목이 뻐근
7. 내일의 작은 행동: 자기 전 한 문장 적기
오늘 고마웠던 것: 동료가 커피를 사줌
```

## 생성된 거울 편지 (humanizer + 결다듬기 통과)
```
민님,

오늘 당신은 이렇게 적었어요. 몸은 무겁고 어깨가 뻐근했고, 한 단어로는 '버팀'.
회의에서 의견이 묻힌 장면이 오래 남았고, 그때 가슴이 조였다고 했어요.
한쪽에선 괜찮은 척하면서, 다른 쪽에선 사실 인정받고 싶다고 적었고요.
그러고는 오늘의 나는 잘 버텼다, 고 끝맺었어요.

'버팀'이라는 한 단어, '잘 버텼다'는 문장, '또 참았네'라는 장면.
오늘 당신의 말에는 버티고 참는 결이 몇 번 지나가요.
(무겁다고만 하기엔, 꽤 오래 어깨로 받치고 있었던 것 같기도 하고요.)

내일은 자기 전에 한 문장을 적어보기로 했지요.
그 문장이 오늘처럼 버틴 이야기일지 아닐지는, 내일의 당신이 적어줄 거예요.

— 이어
```

## 6기준 통과표
| 기준 | 결과 |
|---|---|
| ① 금지어·과장 0 | ✅ 결다듬기 "통과 — 손댈 곳 없음" |
| ② 고객 언어로 정돈 | ✅ '버팀'·'또 참았네'·'잘 버텼다' 원어 유지 |
| ③ 패턴 한 가지, 살짝선 안 | ✅ '버티고 참는 결' 하나 + 괄호 깃털 한 번 |
| ④ 답·조언·평가 없음 | ✅ 처방·원인 단정·감정 단정 없음 |
| ⑤ AI 말투 없음 | ✅ humanizer 통과(본문 em 대시 1개 제거, 사람 신호 유지) |
| ⑥ 윤미선 "나가도 되겠다" | ⏳ 윤미선 최종 판단(seed §8) |

## 고친 점
- humanizer: 본문 가운데 em 대시(—)를 마침표로 바꿈(§14). 서명 "— 이어"는 편지 맺음 관례라 유지.
- 그 외 손댈 곳 없음(편지가 이미 윤미선 결에 가까웠음).

## 실전 1회 — 윤미선 본인 기록 "쏭" (6/23)

> 윤미선이 직접 세션을 하고 자기 기록을 비춰봄(첫 진짜 도착). ⑥ "나에게 도착했나"는 **윤미선이 6/24에 판단 예정.**

### 입력 (실제)
```
하루 끝, 나에게 — 쏭 · 2026.6.23 오후 1:54
3. 감각: 답답함, 가슴이 답답함 / 한 단어: 쉬고 싶다
4. 장면: 너무 열심히 하고 있다는 생각, 쉬는 시간 없이 일하는 / 감각: 턱이 딱딱하다 / 말: 좀 쉬어야 할 거 같아
5. 목소리 A: 더 쉬어야 한다 / 목소리 B: 의미 있는 걸 한다면 괜찮다
6. 한 문장: 쉼과 열심히 사이에서 왔다갔다. / 느낌: (빈칸)
7. 내일의 작은 행동: (빈칸)
고마웠던 것: 목소리 칭찬 / 회사에서 내 일 할 수 있어서 / 자원이 많아서
```

### 생성된 거울 편지
```
쏭님,

오늘 당신은 이렇게 적었어요. 가슴이 답답하고, 한 단어로는 '쉬고 싶다'.
쉬는 시간 없이 일하다 문득 너무 열심히 하고 있다는 생각이 들었고,
그때 턱이 딱딱했다고. 한쪽 목소리는 더 쉬어야 한다고 하고,
다른 쪽은 의미 있는 걸 한다면 괜찮다고 했어요.
그리고 오늘의 나는 쉼과 열심 사이에서 왔다갔다, 라고 적었고요.

'쉬고 싶다'는 한 단어, '쉼과 열심 사이에서 왔다갔다'는 문장, 그리고 두 목소리.
오늘 당신의 말은 쉼과 열심 사이를 여러 번 오가요.
(가슴과 턱은 벌써 '답답하다, 딱딱하다'고 먼저 말해둔 것 같기도 하고요.)

내일 칸은 비워 두셨어요. 비워둔 채로 두어도 돼요.
오늘은 사이에서 왔다갔다 한 하루였다고, 그렇게만 적혀 있어요.

— 이어
```

### 처리 메모
- 6번 느낌·7번 내일 행동이 **빈칸** → 새 행동을 **지어내지 않고**, 비워둔 그대로 비춤(③ 거울이 코치가 안 되는 지점).
- ②비춤 = '쉼과 열심 사이를 오감' 한 가지 + 괄호 깃털 한 번("가슴과 턱이 먼저 말해둔 것 같기도").
- ①~⑤ 통과. ⑥은 윤미선 판단 대기(6/24).
