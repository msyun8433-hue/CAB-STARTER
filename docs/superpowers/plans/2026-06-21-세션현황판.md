# 〈하루 끝〉 세션 현황판 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 폴더의 파일들을 읽어 "오늘 뭐부터?"가 보이는 정적 현황판 HTML을 매번 새로 그려주는 Node 스크립트 하나를 만든다.

**Architecture:** 순수 함수(정규화·분류·판단·렌더)와 부수효과(폴더 읽기·파일 쓰기)를 분리한다. 순수 함수는 `node:test`로 검증하고, `main()`이 파일시스템을 읽어 순수 함수에 넘긴 뒤 결과 HTML을 저장한다. 외부 의존성 없음(Node 표준 모듈만).

**Tech Stack:** Node.js (ESM, `.mjs`), 표준 모듈 `node:fs`·`node:path`·`node:url`·`node:test`·`node:assert`. 빌드·설치 없음.

## Global Constraints

- 외부 npm 패키지 금지 — Node 표준 모듈만 사용 (spec §5)
- macOS 한글 파일명은 `.normalize('NFC')` 로 맞춰 비교 (spec §4)
- `.` 또는 `_` 로 시작하는 파일은 제외 (spec §4)
- 잠듦 기준 = 7일 이상 손 안 댐 (spec §3.2)
- 결과 HTML 경로: `하루끝-키트/현황판.html` (spec §5)
- 데이터 출처: 재료 = `자료함/정리/*.md`, 대본·녹음 = `하루끝-키트/outputs/` (spec §4)
- 금지어 화면에 없을 것: 치유·회복·성장·변화·코칭·명상·소마틱·이완 등 (spec §6)
- 이 폴더는 git 저장소가 아니므로 "커밋" 대신 "저장·실행 확인"으로 갈음
- 보기 전용 — 클릭 편집·상태변경 버튼 없음 (spec §2)

```js
// 〈하루 끝〉 세션 현황판 — 폴더를 읽어 현황판.html 을 매번 새로 그린다.
// 실행: node automations/session-dashboard.mjs  →  하루끝-키트/현황판.html

export const normalizeName = (s) =>
  s.toLowerCase().replace(/[\s,._\-]/g, '');

const scriptKey = (f) =>
  normalizeName(f.replace(/^오디오대본_?/, '').replace(/^세션_?/, '').replace(/\.md$/, ''));

export function classifyOutputs(files) {
  const m4aKeys = files.filter(f => f.endsWith('.m4a')).map(normalizeName);
  const scripts = files.filter(f =>
    f.endsWith('.md') && (f.startsWith('오디오대본') || f.startsWith('세션')));
  const 초안 = [], 녹음 = [];
  for (const s of scripts) {
    const name = s.replace(/\.md$/, '');
    (m4aKeys.some(k => k.includes(scriptKey(s))) ? 녹음 : 초안).push(name);
  }
  return { 초안, 녹음 };
}
```

```js
export const daysSince = (mtimeMs, nowMs) =>
  Math.floor((nowMs - mtimeMs) / 86400000);

export const findStalled = (items, stall = 7) =>
  items.filter(i => i.days >= stall).sort((a, b) => b.days - a.days);

export function decideNextStep({ 초안, 재료 }) {
  const 세션초안 = 초안.find(n => n.startsWith('세션'));
  const pick = 세션초안 || 초안[0];
  if (pick) return `“${pick}” — 대본은 됐어요. 읽고 녹음하면 발행 후보가 됩니다.`;
  if (재료.length) return `정리된 재료 ${재료.length}개가 기다려요. 그중 하나로 질문지·대본을 만들어 보세요.`;
  return `새 상태 하나를 골라 세션을 시작해 보세요.`;
}
```

```js
const ROOT = join(dirname(fileURLToPath(import.meta.url)), '..');
const list = (p) => existsSync(p)
  ? readdirSync(p).map(f => f.normalize('NFC')).filter(f => !f.startsWith('.') && !f.startsWith('_'))
  : [];

function toItems(dir, names, now, stallable = true) {
  return names.map(n => {
    const file = names === undefined ? n : n; // n 은 .md 뗀 이름일 수 있어 원본 파일 다시 찾음
    return n;
  });
}

export function main() {
  const now = Date.now();
  const 정리dir = join(ROOT, '자료함/정리');
  const outDir = join(ROOT, '하루끝-키트/outputs');

  const 재료files = list(정리dir).filter(f => f.endsWith('.md'));
  const outFiles = list(outDir);
  const { 초안: 초안names, 녹음: 녹음names } = classifyOutputs(outFiles);

  const meta = (dir, displayName, originalFile) => {
    const m = statSync(join(dir, originalFile)).mtime;
    return { name: displayName, date: `${m.getMonth() + 1}/${m.getDate()}`,
             days: daysSince(m.getTime(), now), stalled: false };
  };
  const matchFile = (names, display) => names.find(f => f.replace(/\.md$/, '') === display);

  const 재료 = 재료files.map(f => {
    const m = meta(정리dir, f.replace(/\.md$/, ''), f); m.stalled = m.days >= 7; return m;
  }).sort((a, b) => (b.stalled - a.stalled) || (b.days - a.days));

  const 초안 = 초안names.map(d => {
    const m = meta(outDir, d, matchFile(outFiles, d)); m.stalled = m.days >= 7; return m;
  }).sort((a, b) => (b.stalled - a.stalled) || (b.days - a.days));

  const 녹음 = 녹음names.map(d => meta(outDir, d, matchFile(outFiles, d)))
    .sort((a, b) => b.days - a.days);  // 녹음은 잠듦 아님

  const 잠든 = findStalled([...재료, ...초안]);
  const 다음걸음 = decideNextStep({ 초안: 초안names, 재료: 재료files.map(f => f.replace(/\.md$/, '')) });
  const stamp = new Date().toLocaleString('ko-KR', { timeZone: 'Asia/Seoul' });

  const html = renderHtml({ 다음걸음, 잠든, 재료, 초안, 녹음, stamp });
  const outPath = join(ROOT, '하루끝-키트/현황판.html');
  writeFileSync(outPath, html, 'utf8');
  console.log(`현황판 생성: 하루끝-키트/현황판.html`);
  console.log(`  재료 ${재료.length} · 초안 ${초안.length} · 녹음 ${녹음.length} · 잠든 ${잠든.length}`);
}

if (process.argv[1] === fileURLToPath(import.meta.url)) main();
```

> 구현 메모: 위 `toItems`/`matchFile` 는 "표시이름(.md 뗀 것) ↔ 원본 파일명(stat 용)"을 잇기 위한 것. `meta()`에 원본 파일명을 넘겨 mtime을 읽는다. 미사용 `toItems` 헬퍼는 지우고 `matchFile`만 남길 것.
