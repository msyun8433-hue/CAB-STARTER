# 이어거울 실시간 생성 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** HTML 폼 제출 시 Claude API로 거울 글을 실시간으로 생성하고, 같은 페이지에 즉시 표시하면서 동시에 이메일 발송.

**Architecture:** 
- 프론트엔드 (session.html): 폼 제출 → API 호출 → 거울 글 표시
- 백엔드 API (`mirror.js`): 고객 기록 수신 → Claude로 거울 글 생성 → JSON 응답 + 이메일 발송
- Supabase: 기록 + 거울 글 저장 (타임라인용)

**Tech Stack:** 
- Frontend: HTML/JavaScript (Fetch API)
- Backend: Node.js/Express API (Claude SDK)
- Database: Supabase (PostgreSQL)
- Email: Gmail API (기존)

## Global Constraints

- 금지어: 명상·소마틱·치유·회복·성장·코칭 (CLAUDE.md §5)
- 거울 글 톤: 판단/조언/해석 없이, 고객의 단어 정렬 + 마지막 문구만
- 마지막 문단: "평온함을 찾게 하기" (문구는 execution 단계에서 [결다듬기] 스킬로 검토)
- 비용: Claude API 호출 (월 ~$2)

// 1. session 저장
const { data: session, error: sessionError } = await supabase
  .from('sessions')
  .insert([
    {
      user_id: userId,
      recorded_at: new Date().toISOString().split('T')[0],
      sensations,
      one_word: oneWord,
      scene,
      scene_sensation: sceneSensation,
      scene_interpretation: sceneInterpretation,
      voice_a: voiceA,
      voice_b: voiceB,
      one_sentence: oneSentence,
      sentence_feeling: sentenceFeeling,
      tomorrow_action: tomorrowAction,
      gratitude,
    },
  ])
  .select('id')
  .single();

if (sessionError) throw new Error(`Session insert failed: ${sessionError.message}`);
const sessionId = session.id;

// 2. mirror 저장
const { data: mirror, error: mirrorError } = await supabase
  .from('mirrors')
  .insert([
    {
      session_id: sessionId,
      mirror_text: mirrorText,
      metadata: {
        tokens_used: message.usage.output_tokens,
        generated_at: new Date().toISOString(),
      },
    },
  ])
  .select('id')
  .single();

if (mirrorError) throw new Error(`Mirror insert failed: ${mirrorError.message}`);

```javascript
async function submitAndShowMirror() {
  const data = collectFormData();
  
  try {
    const response = await fetch('/api/mirror', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(data),
    });
    
    if (!response.ok) {
      const error = await response.json();
      throw new Error(error.error || 'Mirror generation failed');
    }
    
    const result = await response.json();
    displayMirror(result.mirrorText, result.mirrorId, result.sessionId);
  } catch (error) {
    console.error('Error:', error);
    alert('거울을 생성하는 중에 오류가 발생했습니다: ' + error.message);
  }
}
```

- [ ] **Step 3: 거울 글 표시 UI 작성**

```javascript
function displayMirror(mirrorText, mirrorId, sessionId) {
  // 기존 폼 감추기
  document.getElementById('form').style.display = 'none';
  
  // 거울 섹션 생성/표시
  let mirrorSection = document.getElementById('mirror-section');
  if (!mirrorSection) {
    mirrorSection = document.createElement('section');
    mirrorSection.id = 'mirror-section';
    mirrorSection.style.cssText = `
      padding: 32px 0;
      border-top: 1px solid #e3ddd2;
      background: #faf8f3;
      border-radius: 14px;
      margin-top: 40px;
      padding: 26px;
    `;
    document.querySelector('.wrap').appendChild(mirrorSection);
  }
  
  mirrorSection.innerHTML = `
    <h2>오늘의 거울</h2>
    <div style="
      background: white;
      border: 1px solid #e3ddd2;
      border-radius: 10px;
      padding: 20px;
      line-height: 1.9;
      color: #2b2b28;
      margin: 16px 0;
      white-space: pre-wrap;
      word-break: keep-all;
    ">
      ${escapeHtml(mirrorText)}
    </div>
    <div style="margin-top: 20px; font-size: 14px; color: #9a9489;">
      <p>이 거울이 당신과 함께 있습니다.</p>
      <p><a href="#" onclick="location.reload(); return false;">다시 기록하기</a></p>
    </div>
  `;
  
  // 페이지 상단으로 스크롤
  mirrorSection.scrollIntoView({ behavior: 'smooth' });
  
  // localStorage에 저장 (타임라인용 — Task 4에서 활용)
  localStorage.setItem(`mirror_${sessionId}`, JSON.stringify({
    mirrorId,
    sessionId,
    mirrorText,
    displayedAt: new Date().toISOString(),
  }));
}

function escapeHtml(text) {
  return text
    .replace(/&/g, '&amp;')
    .replace(/</g, '&lt;')
    .replace(/>/g, '&gt;')
    .replace(/"/g, '&quot;');
}
```
