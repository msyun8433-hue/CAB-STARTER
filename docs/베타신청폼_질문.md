# 베타 신청 폼 — 질문 (구글폼에 그대로 붙여넣기)

> 만드는 법: forms.google.com → 빈 양식 → 아래 제목·설명·질문을 그대로 옮긴다.
> 다 만들면 우상단 "보내기 → 링크(🔗)" 주소를 복사 → 노션 페이지 "베타 신청하기" 버튼에 붙인다.

---

## 폼 제목
이어 — 〈하루 끝, 나에게〉 베타 신청

## 폼 설명 (제목 아래)
하루 끝에 자기 자신에게 잠깐 돌아가 보는, 혼자 쓰는 감각·기록 도구입니다.
첫 도구를 함께 써볼 분을 찾습니다. 써보신 뒤 짧은 느낌을 들려주시는 것이 참여 조건입니다.
상담이나 치료가 아닙니다. 불편하면 언제든 멈추어도 괜찮습니다.

---

## 질문

**1. 이름 (또는 불리고 싶은 이름)**
- 유형: 단답형
- 필수

**2. 연락받을 방법** — 도구와 입금 안내를 보내드릴 곳이에요.
- 유형: 단답형
- 필수
- (이메일 또는 카카오톡 아이디)

**3. 요즘, 하루 끝이 어떤가요?** — 편하게 한두 줄이면 됩니다.
- 유형: 장문형
- 선택

**4. 무엇이 궁금해서 신청하셨나요?**
- 유형: 장문형
- 선택

**5. 오디오는 주로 무엇으로 들으실 것 같나요?**
- 유형: 객관식
- 보기: 휴대폰 / 컴퓨터 / 태블릿 / 잘 모르겠음

**6. 워크북은 어느 쪽이 편하세요?**
- 유형: 객관식
- 보기: 인쇄해서 손으로 적기 (PDF) / 컴퓨터로 입력하기 (Word) / 둘 다 받아보고 정하기

**7. 써보신 뒤 짧은 느낌(후기)을 들려주실 수 있나요?** — 베타 참여의 조건이에요.
- 유형: 객관식
- 보기: 네, 들려드릴게요 / 아직 잘 모르겠어요
- 필수

**8. 남기고 싶은 말이 있다면** (자유)
- 유형: 장문형
- 선택
