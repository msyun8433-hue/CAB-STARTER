## 한눈에 보는 여정

```
①발견 ··→ ②맛보기 ──→ ③기본 세션 ──→ ④반복 ──→ ⑤AI 거울 ──→ ⑥한 권
 (미정)    (무료·낮은문턱)  (혼자 완결)    (쌓임)    (2단·고단가)   (vol.쌓임)
```

- **1단(혼자 쓰기)** = ②③④ : 입구. PDF·오디오로 혼자 완결.
- **2단(AI 거울)** = ⑤⑥ : 심장·고단가. 적은 걸 보내면 AI가 정돈·되비춤 → 한 권으로 쌓임.
- 중심은 2단이나 **출시는 1단 먼저**, 2단은 베타 10명 뒤. (★지금-기준.md "쓰는 방식 2층")

## 가격 (가설 — 베타로 검증)

전제: §0 "저볼륨 고단가"(깊이가 값을 만든다) + 꿈의 고객(42세 직장인, 돈 있고 깊이 원함).

| | 1단 (혼자: 오디오+워크북) | 2단 (AI 거울 + 한 권) |
|---|---|---|
| A. 저가 진입형 | ₩9,900~19,000 | 월 ₩29,000 |
| **B. 중가 균형형 ⭐추천** | **₩29,000** | **월 ₩39,000~49,000** |
| C. 고단가 깊이형 | ₩39,000 | 월 ₩69,000 / 분기 ₩150,000 |

- **맛보기는 무조건 무료** (유입·신뢰).
- 추천 = **B.** A는 §0의 "고단가"에 너무 쌈. 진짜 수익은 **2단 월구독**(매달 한 권 쌓이는 값).
- **베타가 별도:** 정가는 B로 두되, 베타 참여자는 **무료 또는 ₩9,900 + 후기 필수**. → §7 "돈 낼 의사 있나?"를 싸게 검증.
- ⚠️ 진짜 가격은 베타 10명이 "얼마면 내겠다"로 정한다. 지금 숫자에 오래 매달리지 말 것.

## 결제 — 사업자등록 없이 가능

| 방법 | 사업자등록 | 카드 | 지금? |
|---|---|---|---|
| **계좌이체 / 카카오·토스 송금** | 불필요 | ✖ | ✅ 오늘 |
| 크몽 (디지털 상품 마켓) | 개인 가능 | 대행 | ✅ (수수료) |
| Gumroad / Lemon Squeezy (해외, 디지털 다운로드) | 불필요 | ✅ | ✅ |
| 네이버 스마트스토어 | 필요 | ✅ | ✖ |

- **베타 = 그냥 계좌이체.** 10명에게 카드 시스템 만들 필요 없음. "입금 확인 → 파일 보냄".
- 카드 자동결제 원하면 → 크몽(국내) 또는 Gumroad(자동 다운로드).
- **사업자등록은 쉬움**(홈택스 무료·온라인·10분). 베타 뒤 "되겠다" 싶으면 그때. 지금 출시의 장애물 아님.
