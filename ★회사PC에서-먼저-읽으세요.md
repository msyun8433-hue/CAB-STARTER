# ★ 회사 윈도우 PC에서 먼저 읽으세요

> 이 USB를 회사 컴퓨터에 꽂고, 이 글을 그대로 따라 하시면 돼요.
> 어려운 말 없이, 순서대로만 하시면 됩니다.

---

## 0. 지금 상황 한 줄 정리
- 내 사업 폴더(`cab-starter`)를 **USB에 통째로 옮겼다.**
- 앞으로 작업은 **맥이든 회사 PC든, 무조건 이 USB 안에서** 한다.
- USB가 **원본(메인)** 이다. 맥 다운로드 폴더에 있는 건 백업일 뿐.

---

## 1. 회사 PC에서 "AI 직원" 켜는 법 (처음 한 번만)

VS Code(편집기)만 깔면 AI는 안 나와요. 아래 4개를 순서대로 하세요.

1. **VS Code 설치**
   (인터넷에 "VS Code 다운로드" 검색 → 윈도우용 설치)

2. **VS Code 안에 "Claude Code" 확장 설치** ← 이게 "AI 직원"이에요
   (VS Code 왼쪽 네모 아이콘(확장) 클릭 → "Claude Code" 검색 → 설치)

3. **Claude 계정 로그인**
   (맥에서 쓰던 그 계정으로 로그인)

4. **USB 안 `cab-starter` 폴더 열기**
   (VS Code → 파일 → 폴더 열기 → USB의 `cab-starter` 선택)

이 4개를 하면 **맥에서 쓰던 것과 똑같은 AI 직원**이 회사 PC에서도 켜져요.

---

## 2. AI가 내 사업을 기억하나요? → 네, 기억해요

- 내 사업 내용(사명·꿈의 고객·금지어·우선순위)은
  폴더 안 `CLAUDE.md` 와 `business` 폴더에 **글로 적혀 있어요.**
- AI는 켤 때마다 이 파일들을 **자동으로 읽어요.**
- 그래서 회사 PC에서 처음 켜도 "윤미선님 사업은 이거구나" 하고 똑같이 알아요.

⚠️ 단, **맥에서 AI랑 나눴던 대화(채팅) 기록 자체는 안 따라와요.**
   (그건 맥 본체에 남아요.) 하지만 중요한 사업 내용은 다 파일에 있으니
   실제 일하는 데는 지장 없어요.

---

## 3. 노션 연결은 "필요할 때만"

- 노션(자료 보관함)에 뭔가 저장·정리를 AI한테 시킬 거면 →
  AI한테 `/mcp` 라고 입력하고 노션 로그인 한 번 (이 PC에서 처음 한 번만).
- 그냥 폴더 안 글쓰기·정리만 할 거면 → **노션 로그인 안 해도 돼요. 건너뛰기.**
- 비밀번호는 USB에 안 담겨 있어서, PC 바꿀 때마다 한 번씩 로그인하는 방식이에요.
  (USB 잃어버려도 노션이 안 털려서 더 안전해요.)

---

## 4. 꼭 지킬 안전 규칙 (이것만!)

1. **작업은 무조건 USB 안에서.**
   맥 백업본이랑 USB를 따로따로 고치면, 나중에 어느 게 최신인지 어긋나요.

2. **USB 뺄 때는 꼭 "안전하게 제거(꺼내기)" 누르고 뽑기.**
   - 윈도우: 화면 오른쪽 아래 USB 아이콘 → "꺼내기"
   - 맥: USB 아이콘 옆 ⏏ 버튼
   - 그냥 뽑으면 파일이 깨질 수 있어요.

3. **비밀번호·고객 정보는 폴더 본문에 적지 않기.** (`.env` 파일에만)
