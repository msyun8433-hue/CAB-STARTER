# 자료함 — 참고자료 던지는 곳

> 정리 안 해도 돼요. **아무거나 막 넣는 방**이에요.
> ⭐ **지금 뭐가 기준인지 헷갈리면 → [★지금-기준.md](★지금-기준.md) 를 먼저 보세요.**

## 여기 넣는 것
- 내 사명·계획·메모 (정리 전이라도 OK)
- 참고할 만한 남의 글·PDF·캡처·링크 모음
- "나중에 쓸지도" 싶은 모든 재료

## AI(나)가 하는 일
- 여기 있는 자료를 **읽고 중요한 내용을 추출**해서
  - 핵심 사업 사실 → `business/` 로
  - 계획서·문서 → `docs/` 로
  정리해 옮겨요.
- **원본은 여기 그대로 남겨둬요.** (지우지 않음 — 언제든 되돌아갈 수 있게)

## 쓰는 법
1. 파일을 이 폴더에 그냥 넣는다 (채팅 첨부 말고 **폴더에**).
2. 나한테 말한다 → "자료함에 넣은 거 읽고 정리해줘" 또는 "이거 참고해서 ○○ 만들어줘".
