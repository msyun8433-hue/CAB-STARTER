# W1 워크북 — 빈 폴더에서 내 사업 매뉴얼 만들기

> 이번 주 한 장이에요. 위에서 아래로 따라가면 돼요. 핵심은 **CLAUDE.md 를 내가 적는 게 아니라, 인터뷰로 사업 파일을 채우면 Claude 가 만들어준다**는 거예요.

## 이번 주 끝나면 손에 남는 것
- **`business/` 안 사업 파일** — 사명·꿈의고객·8부서 지도·자동화 후보·도구 (`/cab-interview` 가 작성)
- **CLAUDE.md** — AI 가 매 세션 읽는 "내 사업 매뉴얼" (`/init` 이 위 파일들을 읽고 생성)
- **자동화 후보 1개** — 다음 주(W2)에 실제로 만들 1순위

> 오늘 약속은 "작동하는 자동화"가 아니라 **"내가 누군지를 파일로 적어두는 것"** 까지예요. 그거면 1주차는 충분해요.

---

## ① 폴더 준비 (1분)
- [ ] 이 폴더 이름을 **내 사업·브랜드 이름**으로 바꿨나요? (예: `미선베이킹`) → 아직이면 [README](../README.md) "받기" 참고
- [ ] 이 폴더엔 **아직 CLAUDE.md 가 없어요. 정상이에요** — ③에서 만들 거예요. (`CLAUDE.md.예시` 는 완성 꼴 참고용)
- [ ] (선택) 머릿속을 먼저 펼치고 싶으면 → [business/마인드맵-가이드.md](../business/마인드맵-가이드.md)

## ② 인터뷰 → 사업 파일 만들기
Claude Code 채팅창에 한 줄:

```
/cab-interview
```

Claude 가 7가지를 물어봐요 (미리 적어두면 더 빨라요):
1. **내 사업 한 줄 + 사명** — 나는 [누구] 의 [어떤 문제] 를 [어떤 방법] 으로 푼다 / 왜 이 일을 하나
2. **꿈의 고객 한 명** — 어떤 상황 + 밤에 잠 못 드는 통증
3. **시간 먹는 반복업무 5~8개** (+ 제일 벗어나고 싶은 통증 1개)
4. **평소 쓰는 도구 전부** (10~25개)
5. **§0 원칙 하나** — 매출·숫자보다 뭘 먼저 보나 (이 사업의 영혼)
6. **내 말투 + 절대 금지어**
7. **우선순위 3개 · 이번 분기 목표 · 안 하는 것**

→ 끝나면 `business/00-사명`·`01-꿈의고객`·`02-8부서`·`03-도구` 가 채워져요. (마인드맵 펼친 게 있으면 그 텍스트를 같이 붙여넣으세요.)

> ③④ 는 8부서·자동화 재료고, 나머지(①②⑤⑥⑦)가 CLAUDE.md 재료예요. 여기에 **`/init` 이 출력·안전·컨텍스트 공통 3섹션을 자동으로 더해** 갖춰진 매뉴얼이 돼요.

## ③ /init → CLAUDE.md 생성
이제 채운 사업 파일을 읽어 매뉴얼을 만들어요:

```
/init
```

Claude 가 `business/` 를 훑고 → **CLAUDE.md(내 사업 매뉴얼)를 만들어 저장**해요. (형식은 `business/_CLAUDE.md-형식.md` 가 잡아줘요. 결과가 `CLAUDE.md.예시` 같은 꼴이면 잘 된 거예요.)

> **안 되면 (폴백)**: `/init` 결과가 "코드 폴더 요약"처럼 나오면 → `business/ 읽고 _CLAUDE.md-형식.md 형식대로 사업 매뉴얼 CLAUDE.md 만들어줘` 라고 한 줄 더.
