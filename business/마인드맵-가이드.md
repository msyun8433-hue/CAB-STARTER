# W1 시작 — 마인드맵으로 내 사업 펼치기

> 빈 칸을 채우는 건 막막해요. 먼저 **머릿속을 마인드맵으로 펼친 다음**, 그걸 Claude Code 에 넘기면 `/cab-interview` 가 8부서·자동화 후보·사업 파일을 만들어요. (CLAUDE.md 는 그다음 `/init` 이 생성해요.)

## 순서 (3단계)

1. **펼치기** — XMind(또는 익숙한 마인드맵 도구)에서 `마인드맵-템플릿.opml` 을 열어요.
   `파일 → 가져오기 → OPML` (XMind 기준). 막막하면 `마인드맵-예시.opml`(태오 사업으로 채운 것)을 먼저 열어 감을 잡아요.
2. **채우기** — 중심 "내 사업" 아래 6가지 가지를 내 사업으로 채워요. 완벽 X, 떠오르는 대로.
3. **넘기기** — 다 채웠으면 `내보내기 → 텍스트`(또는 OPML)로 빼서, Claude Code 채팅에 붙여넣고 `/cab-interview` 를 실행해요.

## 6가지 가지 (왜 이걸 묻나)

| 가지 | 무엇을 | 어디에 쓰이나 |
|---|---|---|
| ① 사업 한 줄 | 누구의 / 어떤 문제 / 어떤 방법 | CLAUDE.md + 8부서 |
| ② 꿈의 고객 한 명 | 상황 + 밤에 잠 못 드는 통증 | CLAUDE.md |
| ③ 반복업무 + 통증 | 시간 먹는 일 5~8개 + 1순위 통증 | 8부서 + 자동화 후보 |
| ④ 쓰는 도구 전부 | 10~25개 | 8부서 도구 배치 |
| ⑤ 톤 · 금지어 | 말투 + AI 가 쓰면 안 되는 표현 | CLAUDE.md |
| ⑥ 우선순위·목표·안 하는 것 | 분기 우선순위·목표 | CLAUDE.md |

①③④ 는 **8부서·자동화** 재료, ②⑤⑥ 은 **CLAUDE.md(내 사업 매뉴얼)** 재료예요. 한 번에 모아서 넘기면 `/cab-interview` 가 `business/` 파일로 저장하고, 그다음 `/init` 이 그걸 읽어 CLAUDE.md 를 만들어요.

> 마인드맵 도구가 없어도 괜찮아요. 종이에 적어 사진을 찍어 붙여넣거나, 6가지를 그냥 글로 적어 붙여넣어도 똑같이 작동해요. 도구는 발산을 돕는 수단일 뿐이에요.
