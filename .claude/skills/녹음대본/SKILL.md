# 녹음 대본 (〈하루 끝, 나에게〉 오디오 세션)

미선이 **소리 내 읽어 녹음할** 오디오 세션 대본을 만든다. 완성품을 흉내내는 게 아니라, **하루 끝의 "상태"를 넣으면 녹음할 대본 초안이 나오게** 하는 것.
가장 가까운 본보기: `하루끝-키트/outputs/오디오대본_하루끝-나에게_v0.4.md` (형식·말투를 여기에 맞춘다). 결 기준: `하루끝-키트/seeds/yunmiseon_seed.md` · `CLAUDE.md`.

## 무엇을 받나
다음 중 무엇이든. 비면 **한 가지만** 묻는다("어떤 하루 끝의 상태로 만들까요? 한 줄이면 충분해요").
- 하루 끝의 **상태** 한 줄~몇 줄 (예: "오랜만에 혼자가 된 주말, 뭘 해야 할지 몰라 멍한 오후")
- 주제, 또는 `질문과제정리`가 뽑아둔 **질문 후보 묶음**
- 폴더 안 파일 경로(예: `하루끝-키트/inputs/...`)

## 가장 중요한 원칙: 거울 (YOU MUST)
이 도구는 **답·조언·해석·위로·교훈을 주지 않는다.** 듣는 사람이 *스스로* 자기에게 닿게 하는 **거울**이다.
- 하지 말 것: "괜찮아질 거예요", "이렇게 해보세요", "당신은 충분합니다", 어떤 평가·진단·정답.
- 할 것: 감각을 묻고, 장면을 떠올리게 하고, **머무를 시간(멈춤)** 을 주고, 적게 한다.
- 이유: §0 — 사람을 *바꾸는* 게 아니라 **자기 자신과의 대화를 이어가게** 한다. 미선이 안 끼고도 혼자 굴러가는 도구여야 한다.

## 형식 (v0.4 그대로)
**읽는 것 / 안 읽는 것**을 명확히 나눈다.
- **소리 내 읽는다:** 줄글(문단)만.
- **절대 안 읽는다:** `## [m:ss] 제목` 줄 · `[멈춤 N초]` · `(지문)`.
- **워크북 안내는 칸 이름으로.** 번호를 말하지 않는다(틀릴 위험·덜 담백). 예: "워크북 **감각 칸**에 적어 둡니다."

각 섹션은 이렇게:
```
## [m:ss] 섹션이름
(필요하면 지문)
줄글 한두 문단. 소리 내 읽는 부분.

[멈춤 N초]
```

## 흐름 (이 리듬 · 상태에 맞게 살을 붙인다)
1. **문턱** `[0:00]` — 십 분 동안 멈춘다는 안내. "고치는 시간이 아니라 들여다보는 시간." 워크북·펜 곁에. *불편하면 멈춰도 된다(상담·치료 아님 안내)*. 역할 내려놓기.
2. **숨·몸** `[~1:20]` — 눈 감기(선택), 숨 한 번, 몸이 닿은 곳으로. [멈춤 길게]
3. **감각** `[~2:40]` — 지금 몸에서 먼저 느껴지는 것 → 한 단어 → **감각 칸**에 적기.
4. **오래 남은 장면** `[~4:00]` — 오늘을 되감아 이상하게 남는 장면 하나 → 그때의 몸 → **오래 남은 장면 칸**.
5. **두 목소리** `[~5:40]` — "나는 지금…" / "하지만 사실 나는…" 둘 다 적기. 하나를 이기게 하지 않는다.
6. **(상태 맞춤 1~2 섹션)** — 받은 상태에 맞춰 결이 닿는 질문·머무름을 1~2개 더. (상태가 '멍한 주말'이면 그 결로)
7. **한 문장 남기기** `[~8:30]` — 오늘의 나에게 건네는, 짧은 한 문장을 **마무리 칸**에 적기.
8. **문 닫기 / 마무리** `[~9:40]` — 천천히 돌아오기. "오늘의 나를 이만 내려놓습니다." 끝인사("안녕히 주무세요" 등). *뒤에 한 박자 쉬고 녹음 정지* 지문.

길이 약 **10~11분**. [멈춤]을 넉넉히 둬서 듣는 사람이 실제로 쓰고 적을 시간을 준다.

## 말투·단어
- **말투:** ~합니다 / ~봅니다. 낮고 담백하게. 명령(~하세요)보다 **권유(~봅니다, ~해도 좋습니다)**.
- **단어:** 추상어보다 **구체**. 몸·장면·사물(발바닥, 창에 비친 얼굴, 답하지 못한 문자).
- **여백:** 결론을 서둘러 내리지 않는다. 감상 과잉 금지.

## 절대 금지어 (YOU MUST — 빼고 일어나는 일로)
- 장르 이름: 명상 · 소마틱 · 바디스캔 · 이완 · 마음챙김 · 호흡법 → 숨 · 몸 · 무게 · 온도 · 머무름
- 개념어: 치유 · 회복 · 성장 · 변화 · 자기계발 · 코칭 · 클래스 · 교육
- 과장·압박: "괜찮아질 거예요" 류 약속·동기부여. (거울 원칙과 겹친다)
- 대본 완성 후 한 번 더 훑어 금지어가 없는지 점검한다. (필요하면 `결다듬기` 와 같은 기준)

## 어디에 저장하나
- `하루끝-키트/outputs/오디오대본_<상태짧게>_v0.1.md` 로 저장하고, 채팅에도 보여준다.
- 맨 위에 **녹음 팁**(천천히·낮게, [멈춤]에서 진짜 쉬기, 번호 말 안 함, 끝에 한 박자 쉬고 정지)과 **읽는 법**(줄글만 읽기)을 붙인다.
- 버전은 v0.1부터. 미선이 고치면 v0.2…

## 미선이 직접 하는 것
- 사람에게 닿는 **최종 문장**과 어떤 상태를 낼지는 미선이 고른다. 대본은 *초안*이다.
- 톤이 미세하게 어긋나면, 본보기 v0.4의 그 문단을 다시 읽고 맞춘다.
