# /dream — 메모리 통합

세션의 활동, 결정, 피드백을 메모리 파일로 저장하여 다음 기기·다음 세션에서 빠르게 참고할 수 있게 합니다.

## 너(AI)가 할 일

1. **dream 스킬을 즉시 호출** — `Skill` 도구로 `dream` 을 호출
2. **스킬 지시를 끝까지 따르기** — 4단계 프로세스(방향 설정 → 신호 수집 → 통합 → 정리)를 모두 실행
3. **메모리 파일 생성 또는 갱신** — 요청한 타입(`user`/`feedback`/`project`/`reference`)의 메모리 파일 작성
4. **MEMORY.md 인덱스 갱신** — 새 메모리를 인덱스에 한 줄로 추가

## 사용 가능한 타입

```
/dream user       # 역할, 선호도, 지식을 메모리에 기록
/dream feedback   # 이번 대화의 지시·피드백을 메모리에 기록
/dream project    # 프로젝트 진행 상황·결정을 메모리에 기록
/dream reference  # 외부 도구·문서 위치를 메모리에 기록
```

## 메모리 저장 위치

`~/.claude/projects/-Volumes-NO-NAME----CAB-STARTER-/memory/`

- `MEMORY.md` — 전체 색인 (한 줄 요약)
- `user_*.md` — 사용자 역할·선호도·지식
- `feedback_*.md` — 사용자 지시·피드백
- `project_*.md` — 프로젝트 진행·결정
- `reference_*.md` — 외부 리소스 위치

## 중요 규칙

- **이어가기.md 우선** — 현재 상황의 기준
- **상대 날짜 → 절대 날짜** — "지난주" → "2026-06-21"로 변환
- **중복 제거** — 기존 메모리와 병합 여부 확인
- **추측 금지** — 대화·로그에 명시된 것만 기록
