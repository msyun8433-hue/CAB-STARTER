# _예시-콘텐츠-자동화 (복사용 템플릿)

이 폴더는 **자동화 하나가 어떻게 생겼는지** 보여주는 예시예요. 실제로 돌지는 않아요.

## 쓰는 법
1. 이 폴더를 통째로 복사 → 이름을 내 자동화로 바꿔요 (예: `견적서-자동화`)
2. 안의 `CLAUDE.md` 의 [ ] 자리를 채워요 (7가지 구성)
3. W2(`/cab-w2-tools`)에서 이 폴더를 기준으로 실제 자동화 1개를 만들어요

> 폴더 이름 앞 `_` 는 "예시·템플릿"이라는 표시예요. 내 진짜 자동화엔 `_` 빼고 만들어요.

## 왜 자동화마다 CLAUDE.md 가 또 있나요?
- 맨 위 폴더 CLAUDE.md = **사업 전체** 매뉴얼 (내가 누군지·말투·금지)
- 자동화 폴더 CLAUDE.md = **이 자동화에만** 필요한 규칙 (7가지 구성)
- 둘은 위아래로 쌓여요. 위는 자동 적용, 여긴 더할 것만. 그래서 짧게 유지돼요.
