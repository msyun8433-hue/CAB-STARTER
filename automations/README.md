# 자동화 (automations/)

실제로 도는 자동화를 **하나당 폴더 하나**로 여기에 둬요.

- 각 폴더 안에 그 자동화만의 `CLAUDE.md`(7가지 구성)를 둬요 — 사업 전체 매뉴얼 위에 쌓여요
- `_예시-콘텐츠-자동화/` 를 복사해서 시작하세요 (`_` 는 예시 표시)
- W3 에서 만든 에이전트 1개도 여기에 저장돼요

> 자동화 **설계서·기록**은 `business/04-자동화/` 에, **실제 도는 폴더**는 여기에.
> CLAUDE.md 3층: 전역(`~/.claude`) → 사업 전체(맨 위 CLAUDE.md) → 자동화 폴더(여기 각 폴더).

## 파이썬 같은 코드 자동화는?

여기 둬요. 단, 헷갈리지 마세요 — **파이썬 스크립트는 "내 자동화 자산"이지 Claude Code 의 "Tool" 이 아니에요.**
- Claude 는 그 스크립트를 **`Bash` 라는 내장 Tool 로 실행**해요 (`python 정리.py`)
- 즉 *코드는 내가 만들고*, *그걸 돌리는 손(Bash·MCP)은 이미 쥐고 있어요*
- 그래서 `.claude/` 가 아니라 여기(`automations/`) 서랍에 둬요
